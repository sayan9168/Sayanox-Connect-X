# 🚀 Sayanox Connect X
**The Future of Social Networking: High-Speed, High-Security, & Zero-Boring Features.**

Sayanox Connect X is a next-generation social web application that merges the engagement of TikTok's Reels with the seamless communication of Instagram, all wrapped in a high-security shell.

## ✨ Advanced Features
- 🎬 **Infinite Reels:** Smooth vertical scrolling with Framer Motion and Edge Video Streaming.
- 🔐 **Sentinel-Auth:** Fingerprint & Biometric gateway separate from device lock.
- 💬 **Whisper Protocol:** E2EE (End-to-End Encrypted) messaging with Vanish Mode.
- ⚡ **Developer Mode:** In-browser code snippet execution & terminal-style UI options.
- 🌍 **Global Edge Hosting:** Optimized for Vercel with serverless functions.

## 🛠 Tech Stack
- **Frontend:** Next.js 15 (App Router), Tailwind CSS, Framer Motion.
- **Backend:** Supabase (Real-time DB), Node.js / Rust Edge Functions.
- **Security:** AES-GCM Encryption, SentinelGuard-SDK.
- **Media:** Cloudinary / Mux for Video Infrastructure.

## 🚀 Deployment
Deployed and optimized for [Vercel](https://vercel.com).
